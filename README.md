# Meu Portfólio Git

Este repositório foi criado para a atividade EI1 da disciplina de Desenvolvimento de Software, no CIn-UFPE.

A proposta da atividade foi praticar o uso do Git de forma mais organizada, trabalhando com commits semânticos, branches, pull requests e resolução de conflito. O foco aqui não é fazer um projeto grande, e sim mostrar o processo de versionamento e colaboração ao longo dos exercícios.

## Estrutura do repositório

- `README.md` — apresentação do repositório
- `REFLEXAO.md` — reflexão pessoal sobre o que foi aprendido
- `exercicios/ex01-commits/` — evidências do exercício de commits
- `exercicios/ex02-branches/` — evidências do exercício de branches
- `exercicios/ex03-conflito/` — registro da criação e resolução de conflito
- `projeto/hello-world/` — mini-projeto simples usado durante a atividade

## O que foi praticado

Durante esta atividade, pratiquei:

- criação de commits com padrão Conventional Commits
- uso de branches com propósitos diferentes
- abertura e merge de pull requests no GitHub
- criação de conflito real e resolução manual
- organização de um repositório com histórico mais legível
